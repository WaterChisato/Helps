隐私政策

更新日期：2025-01-01

生效日期：2025-01-01

本隐私政策旨在向您说明我们如何收集、使用、存储、共享及保护您的个人信息。 我们深知个人信息对您的重要性，并会尽最大努力保护您的隐私安全。 请您在使用我们的服务前，仔细阅读并理解本政策内容。

如果您使用了此网站，则默认同意本隐私政策的全部内容。

一、我们收集的信息

1. 账户注册信息

  • 昵称、头像、手机号、邮箱、密码（加密存储）
  • 第三方登录授权信息（如微信、QQ、Apple ID）

2. 设备信息

  • 设备型号、操作系统版本、唯一设备标识符（如 IMEI/IDFA/Android ID）
  • 网络类型、IP 地址、运营商信息

3. 日志信息

  • 访问时间、访问路径、页面停留时长
  • 系统崩溃日志、错误日志、性能数据

4. 使用行为信息

  • 功能使用记录、点击行为、搜索记录
  • 偏好设置、语言选择、通知设置

5. 业务相关信息

  • 订单信息、支付记录（不含敏感支付数据）
  • 售后记录、客服沟通记录

6. Cookie 与本地存储信息

  • 会话状态、登录状态、偏好设置
  • 用于统计分析的匿名标识符

7. 法律要求或经您授权的其他信息

二、我们如何使用信息

  • 提供、维护、优化我们的服务
  • 用于身份验证、安全防护、反欺诈监测
  • 改进产品体验、进行数据分析与统计
  • 向您发送服务通知、更新提示等必要信息
  • 在获得授权的情况下，用于个性化推荐或营销活动
  • 履行法律法规要求的义务

三、我们如何共享、转移和公开披露信息

1. 共享

我们不会向任何无关第三方共享您的个人信息，除非：

  • 事先获得您的明确授权
  • 为实现核心功能或提供服务所必须（如云服务提供商）
  • 法律法规要求或政府机关依法提出请求
  • 为维护用户或公众安全、合法权益所必须

2. 第三方 SDK（如适用）

  • 示例：统计分析 SDK — 用于分析用户行为，提升产品体验
  • 示例：推送 SDK — 用于向您发送通知

3. 转移

如涉及业务合并、收购、资产转移，我们会要求新的持有方继续受本政策约束。

4. 公开披露

我们不会公开披露您的个人信息，除非：

  • 获得您的明确授权
  • 基于法律法规要求

四、跨境传输

在某些情况下，我们可能需要将您的数据传输至境外服务提供商（如云服务、内容分发网络）。 我们会提前向您告知传输目的、数据类型、接收方信息，并采取加密、访问控制等措施确保安全。

五、我们如何存储和保护信息

  • 使用加密、访问控制、日志审计等措施保护数据安全
  • 仅在实现目的所需的最短期限内存储您的个人信息
  • 数据到期后进行删除或匿名化处理

安全事件通知

如发生数据泄露、损毁等安全事件，我们会按照法律要求通过邮件、短信、公告等方式通知您， 并向监管机构上报。

六、您的权利

  • 访问、复制您的个人信息
  • 更正、补充您的个人信息
  • 删除您的个人信息（法律法规另有规定除外）
  • 撤回授权或注销账户
  • 限制处理权、数据可携权（如适用）

七、未成年人保护

我们高度重视未成年人的个人信息保护。 如您为未成年人，请在监护人同意后使用我们的服务。

八、政策更新

我们可能会根据业务变化或法律要求更新本政策。 重大变更将通过公告或弹窗方式通知您。

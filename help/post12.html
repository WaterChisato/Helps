🌊 漂流瓶系统使用说明

本系统为 纯 PHP + MySQL 网页版漂流瓶，支持匿名扔瓶子、捡瓶子，并提供后台管理功能。 所有内容均由管理员人工审核，若发现广告或违规内容将被删除。

一、系统用途

漂流瓶系统适用于：

  • 匿名吐槽、树洞、留言墙
  • 轻量级互动社区
  • 匿名心情记录
  • 开源学习 PHP + MySQL 项目结构

二、前台功能

  • 扔瓶子：用户输入内容后提交（普通用户每天最多 3 次）
  • 捡瓶子：随机展示数据库中的一条内容
  • 自适应布局：手机和电脑均可完美显示

三、后台功能

  • 管理员登录（默认账号：admin / 114514）
  • 管理漂流瓶（查看 / 删除）
  • 修改管理员用户名和密码
  • 退出登录

四、管理员如何判断“广告内容”？

本系统由管理员人工审核，以下内容将被视为广告并删除：

1. 含有链接的内容

  • 任何 http:// 或 https:// 开头的内容
  • 短链接（如 t.cn、bit.ly、tinyurl 等）
  • 推广链接、邀请链接、下载链接

2. 明显推广或营销内容

  • “关注公众号”
  • “扫码加我”
  • “优惠、返利、代理、推广”
  • “高薪兼职、日赚 XX、赚钱项目”

3. 含联系方式的内容

  • 手机号（11 位数字）
  • QQ 号（5–12 位数字）
  • 微信号（字母数字组合）
  • 群号、客服号、商务合作号

4. 重复刷屏内容

  • 同一用户（同 IP）短时间内重复发相似内容
  • 明显机器人行为

五、管理员处理方式

管理员在后台 manage_bottles.php 中可以：

  • 查看所有用户提交的瓶子
  • 一键删除广告或违规内容
  • 保持漂流瓶内容干净、健康

六、总结

本系统提供一个轻量、匿名、干净的留言平台。 管理员通过人工审核方式删除广告和违规内容，确保用户体验良好。 若你在使用中发现垃圾广告，只需在后台删除即可。

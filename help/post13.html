次放漂流瓶隐私政策

欢迎使用由 @WaterChisato 开发的“次放漂流瓶”机器人（以下简称“本服务”）。我们深知个人信息对您的重要性，并承诺尽力保护您的隐私。请在使用前仔细阅读以下条款：

1. 我们收集的信息

为了保障系统的稳定运行及内容安全，我们仅收集实现功能所必需的最少信息：

  • 身份唯一标识：我们会记录您的 Telegram UID 或 QQ 平台分配的 OpenID。这些信息在数据库中会带有 telegram_ 或 qq_ 前缀。
  • 投掷内容：系统会存储您通过 /throw 指令主动发送的文本消息。
  • 时间戳：为了实现频率限制，系统会记录您最近一次投掷的时间。

2. 数据的使用规则

  • 频率限制机制：为了防止恶意灌水并保护服务器资源，系统设置了“每人每日限投掷 3 次”的硬性限制。您的 ID 信息仅用于校验该限制是否触发。
  • 内容分级审计：系统会对您发送的内容进行自动化扫描，将其标记为“正常”、“疑似”或“广告”。
  • 管理员管理权：站长通过后台管理界面，拥有对违规数据（如广告内容）进行单向删除的最终操作权。

3. 数据的安全保护与境外传输

我们承诺不会将您的交互数据出售给第三方。由于技术部署原因（如使用 Cloudflare 中继技术解决连接问题），您的数据可能在加密状态下经过境外服务器传输。

详情请点击 这里 查看传输境外许可。

4. 赞助数据的处理（仅限外部页面）

虽然 Telegram 机器人当前未集成赞助功能，但若您通过外部 Web 页面进行自助赞助：

  • 我们将通过后台人工审核流程比对您的支付单号与用途说明。
  • 审核通过后的记录将作为“赞助列表名单”展示在对应的帮助界面中。

5. 联系与撤回

如果您希望删除您的相关记录或有任何疑问，请通过官方渠道联系我们：

联系邮箱：feedback@waterchisato.top
